## 9. Photos — shot-list (upload 20+ now, add a few weekly)
Profiles with lots of photos get **~5× more calls**. Aim for 20+ at launch:

- **Headshot** — Joshua, professional (you already have `/headshot.webp`)

- **Team** — any group/working shots
- **Sold signs / "Just Sold"** — 4–6, ideally with happy clients (get permission)
- **Closing table / handing over keys** — 2–3
- **Neighborhood/landmark shots** — a few per top market: Franklin Main Street, Cool Springs, Downtown Brentwood, The Gulch/Nashville, Spring Hill, Murfreesboro square. These reinforce service-area relevance.
- **At-work** — showing a home, open house, signing paperwork
> Geotag/name files descriptively before upload (e.g., `franklin-tn-main-street.jpg`). Add a couple of fresh photos weekly — recency is a ranking signal.
